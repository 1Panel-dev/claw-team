# Source Install

[中文版](./source-install.zh-CN.md)

1. Clone the repository.

```bash
git clone https://github.com/1Panel-dev/claw-team.git
```

2. Enter the `channel/` directory.

```bash
cd claw-team/channel
```

3. Install dependencies.

```bash
npm install
```

4. Build the plugin.

```bash
npm run typecheck
npm test
npm run build
```

5. Install the plugin.

```bash
openclaw plugins install -l .
```

If you see missing config errors for `baseUrl`, `outboundToken`, or `inboundSigningSecret` at this step, do not stop. Continue to the config step.

6. Enable the plugin.

```bash
openclaw plugins enable claw-team
```

7. Open the Claw Team client, go to the `OpenClaw` page, and create a new instance or edit an existing one.

8. Fill in these values in the instance drawer:

- `OpenClaw URL`
  Enter the current OpenClaw instance URL.
- `Gateway Token`
  Enter the token currently used by the OpenClaw Gateway.

9. Save the instance first.

After the instance is saved successfully, Claw Team will generate:

- `outboundToken`
- `inboundSigningSecret`

Only after that will the `OpenClaw JSON Config` become available to copy.

10. Click the copy icon next to `OpenClaw JSON Config` in the instance drawer.

The client will generate the actual config snippet for:

- `skills`
- `channels.claw-team.accounts.default.baseUrl`
- `outboundToken`
- `inboundSigningSecret`
- `gateway.baseUrl`

You only need to fill in `Gateway Token`. The rest is generated by Claw Team.

11. Open the OpenClaw config file.

Common path:

```text
~/.openclaw/openclaw.json
```

12. Merge the copied JSON snippet into `openclaw.json`.

Notes:

- Do not overwrite the whole file.
- If `skills` or `channels` already exist in `openclaw.json`, review and merge them carefully by hand.

13. Restart the Gateway.

```bash
openclaw gateway restart
```

If you are running OpenClaw in a container, restart the OpenClaw container instead of using `openclaw gateway restart`.

14. Verify the installation.

```bash
openclaw plugins list
openclaw plugins inspect claw-team
openclaw skills list
```
