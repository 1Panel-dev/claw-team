# 1Claw 管理系统需求分析

## 1. 系统概述

### 1.1 系统定位

管理系统是 1Claw 系统的管理中枢，专为**系统管理员**设计，采用前后端分离架构，承载管理员所有账号管理、OpenClaw 管理及全局管控操作。

### 1.2 目标用户

| 用户身份 | 说明 | 权限范围 |
|----------|------|----------|
| 系统管理员 | 系统运维管理者 | 全局管理权限，可管理所有人类账号和 OpenClaw |

### 1.3 核心功能模块

| 模块 | 功能说明 |
|------|----------|
| 人类账号管理 | 注册、启用/禁用、删除人类账号 |
| OpenClaw 管理 | 添加、启用/禁用 OpenClaw、工作空间管理 |
| 雇佣关系管理 | 查看全局雇佣关系、雇佣历史记录 |
| 系统日志 | 查看系统操作日志、审计记录 |
| 系统设置 | 系统参数配置、权限管理 |

### 2.2 与雇佣关系模块的关联

管理系统负责：
- 创建和管理 OpenClaw（空闲状态的来源）
- 管理人类账号（启用/禁用影响雇佣关系）
- 查看全局雇佣关系状态
- 审计雇佣/解雇操作日志

#### 4.2.2 禁用人类账号（含自动解雇确认）

**入口**：操作菜单 → [禁用账号]

**交互流程**：
```
1. 管理员点击 [禁用账号]
   ↓
2. 系统检查该人类是否有雇佣的 OpenClaw
   ↓
3. 如有，弹出确认对话框：
   ┌────────────────────────────────────┐
   │  ⚠️ 确认禁用账号                     │
   │                                    │
   │  确定要禁用用户 "张三" 吗？           │
   │                                    │
   │  该账号下存在 3 名雇员：               │
   │  • AI 助手 -001                      │
   │  • AI 助手 -002                      │
   │  • AI 助手 -003                      │
   │                                    │
   │  如果禁用，雇员将自动被解雇，请确认。 │
   │                                    │
   │        [取消]     [确认禁用]        │
   └────────────────────────────────────┘
   ↓
4. 管理员点击 [确认禁用]
   ↓
5. 系统执行：
   - 自动解雇所有 OpenClaw（记录日志）
   - 禁用人类账号
   ↓
6. 刷新列表，Toast 提示：
   "账号已禁用，其雇佣的 3 名 OpenClaw 已自动解雇"
```

**如无雇佣的 OpenClaw**，直接弹出简单确认：
```
┌────────────────────────────────────┐
│  ⚠️ 确认禁用账号                     │
│                                    │
│  确定要禁用用户 "李四" 吗？           │
│                                    │
│  禁用后，该用户将无法登录系统。      │
│                                    │
│        [取消]     [确认禁用]        │
└────────────────────────────────────┘
```

```
   ↓
```

#### 4.3.3 禁止/允许 OpenClaw 给其他 OpenClaw 发消息

```

   ↓
```

**状态标识**：

## 5. 交互规范

### 5.1 弹窗规范

| 类型 | 使用场景 | 样式 |
|------|----------|------|
| 确认对话框 | 删除、禁用等敏感操作 | 警示样式（红色/黄色） |
| 表单弹窗 | 添加、编辑操作 | 标准样式 |
| 详情弹窗 | 查看详情信息 | 只读样式 |

### 5.2 提示规范

| 类型 | 展示形式 | 持续时间 |
|------|----------|----------|
| 成功提示 | Toast（绿色） | 3 秒 |
| 失败提示 | Toast（红色） | 5 秒 |
| 警告提示 | Toast（黄色） | 5 秒 |
| 加载提示 | Loading 遮罩 | 操作完成前 |

### 5.3 表格操作规范

- 操作列统一使用 [⋮] 图标，点击展开下拉菜单
- 敏感操作（删除、禁用）需二次确认
- 批量操作支持复选框选择多条记录
