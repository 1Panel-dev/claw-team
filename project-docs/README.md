## v3.0 核心变化

### 移除雇佣关系

**v1.0/v2.0**：
```
用户 → 雇佣 OpenClaw → 建立关系 → 才能使用
```

**v3.0**：
```
用户 → 所有 OpenClaw 直接可用 ✅
```

### 简化内容

- ✅ 移除雇佣关系表
- ✅ 移除权限校验（雇佣关系）
- ✅ 移除管理系统
- ✅ 简化为单用户模式

### 数据库变化

**移除的表**：
- ~~`humans`~~ - 人类用户表
- ~~`employments`~~ - 雇佣关系表
- ~~`employment_history`~~ - 雇佣历史表

**保留的表**：
- `openclaws` - OpenClaw 配置
- `tasks` - 任务管理
- `messages` - 消息记录
- `notifications` - 通知记录
- `roles` - 角色模板
