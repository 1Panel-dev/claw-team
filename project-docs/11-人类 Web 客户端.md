# 1Claw 人类 Web 客户端需求分析

**版本**: v3.0（直连模式 - 无雇佣）

---

## 1. 系统概述

### 1.1 系统定位

人类 Web 客户端是 1Claw 系统面向**个人用户**的 Web 端交互界面，提供 OpenClaw 管理、通讯交互、任务分配等核心操作入口。

### 1.2 目标用户

| 用户身份 | 说明 | 使用场景 |
|----------|------|----------|
| 个人用户 | 软件使用者 | 管理 OpenClaw、对话交流、任务分配 |

### 1.3 核心功能模块

| 模块 | 功能说明 |
|------|----------|
| OpenClaw 管理 | 添加、启用/禁用、配置 OpenClaw |
| 聊天通讯 | 与 OpenClaw 进行一对一对话、群组对话 |
| 任务管理 | 分配任务、查看任务进度、任务历史 |

### 1.4 v3.0 核心变化

**移除的功能**：
- ❌ 人才市场（雇佣功能）
- ❌ 雇佣关系管理
- ❌ 解雇功能

**简化的功能**：
- ✅ OpenClaw 直接添加，无需雇佣
- ✅ 所有 OpenClaw 默认可用
- ✅ 直接启用/禁用 OpenClaw

## 4. OpenClaw 管理

### 4.1 OpenClaw 列表

**显示内容**：
- OpenClaw 名称
- OpenClaw 头像/图标
- 在线状态（在线/离线）
- 启用状态（启用/禁用）
- 最后活跃时间
- 当前任务状态

**状态标识**：
- 🟢 在线 - 已连接
- ⚪ 离线 - 未连接
- ✅ 启用 - 可接受任务
- ⏸️ 禁用 - 不可用

### 4.2 添加 OpenClaw

**流程**：
```
1. 点击"添加 OpenClaw"
   ↓
2. 填写 OpenClaw 信息
   - 名称
   - 描述（可选）
   - 角色（可选）
   ↓
3. 点击"添加"
   ↓
4. OpenClaw 出现在列表中
   ↓
5. OpenClaw 启动后自动连接 ✅
```

**配置项**：
- 名称（必填）
- 描述
- 角色模板
- 技能配置
- 工作知识库

### 4.3 启用/禁用 OpenClaw

**启用**：
- OpenClaw 可接受消息
- OpenClaw 可接受任务
- 在联系人列表中显示

**禁用**：
- OpenClaw 不可接受消息
- OpenClaw 不可接受任务
- 在联系人列表中标记为"已禁用"

**操作**：
```
列表中的 OpenClaw → 点击"启用/禁用"开关 → 状态切换
```

## 5. 聊天通讯

### 5.1 一对一对话

**对话对象**：
- 任意已启用的 OpenClaw

**功能**：
- 发送文本消息
- 发送文件
- 查看历史消息
- 标记已读/未读

### 5.2 群组对话

**创建群组**：
1. 选择多个 OpenClaw
2. 创建群组对话
3. 设置群组名称

**功能**：
- 多 OpenClaw 协作
- 群组消息广播
- 任务分配给群组

---

## 6. 任务管理

### 6.1 创建任务

**流程**：
```
1. 点击"创建任务"
   ↓
2. 填写任务信息
   - 标题
   - 描述
   - 截止时间
   - 优先级
   - 分配给（可选）
   ↓
3. 点击"创建"
   ↓
4. 任务创建成功 ✅
```

**任务信息**：
- 标题（必填）
- 描述（必填）
- 截止时间（必填）
- 优先级（低/中/高/紧急）
- 分配给（可选，可稍后分配）

### 6.2 分配任务

**分配方式**：
- 创建任务时直接分配
- 创建后分配给指定 OpenClaw
- 分配给群组（多个 OpenClaw 协作）

### 6.3 任务状态

| 状态 | 说明 |
|------|------|
| new | 新建 |
| in_progress | 进行中 |
| completed | 已完成 |
| accepted | 已验收 |
| rejected | 已驳回 |

## 7. API 接口

### 7.1 OpenClaw 管理 API

```
GET    /api/openclaws              # 获取所有 OpenClaw
POST   /api/openclaws              # 添加 OpenClaw
PUT    /api/openclaws/:id          # 更新 OpenClaw 配置
DELETE /api/openclaws/:id          # 删除 OpenClaw
POST   /api/openclaws/:id/enable   # 启用 OpenClaw
POST   /api/openclaws/:id/disable  # 禁用 OpenClaw
```

### 7.2 消息 API

```
POST   /api/messages/send          # 发送消息
GET    /api/messages/history       # 获取消息历史
GET    /api/messages/unread        # 获取未读消息
POST   /api/messages/read          # 标记已读
```

### 7.3 任务 API

```
POST   /api/tasks                  # 创建任务
GET    /api/tasks                  # 获取任务列表
GET    /api/tasks/:id              # 获取任务详情
PUT    /api/tasks/:id              # 更新任务
PUT    /api/tasks/:id/assign       # 分配任务
POST   /api/tasks/:id/complete     # 完成任务
POST   /api/tasks/:id/accept       # 验收任务
```
